# ProtectEye 护眼卫士
**一款运行于 Windows 系统托盘的轻量级、专业护眼提醒工具。**

## 🌟 核心特性
- **微休息与深度休整**：严格遵循 20-20-20 护眼法则（微休息），并支持自定义工作与深度休息周期。
- **智能免打扰 (DND)**：自动感知全屏应用（游戏、会议、演示），智能推迟休息提醒，不打扰关键任务。
- **闲置智能重置**：感知用户离开电脑状态，自动重置计时，无需手动操作。
- **多主题支持**：内置“暗夜极客黑”、“冷翡高级灰”、“复古纸墨色”三款精致主题，完美适配不同审美。
- **用眼分析**：实时统计每日工作与休息时长，基于 CVS/DES 模型评估视疲劳指数，辅助养成健康习惯。
- **全屏沉浸休息**：提供全屏极光呼吸动效与随机古诗词/健康贴士，让休息也成为一种享受。

## 🛠️ 技术栈
- **核心框架**: .NET 8.0 / WPF
- **系统底层**: Win32 API (P/Invoke)
- **数据序列化**: System.Text.Json

## 🚀 快速开始
1. 下载最新的 Release 版本。
2. 运行 `ProtectEye.exe`。
3. 在系统托盘双击图标打开设置，开启健康用眼之旅。
